# Data Sources

Synthetic source-style data for K-12 learning outcomes dashboard analytics.

- `entities.csv`: 36 district metric records.
- `daily_metrics.csv`: 5,040 district metric-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
